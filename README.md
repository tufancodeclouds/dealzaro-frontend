# Dealzaro frontend

## Tech Stack

- React
- Vite
- Tailwind CSS
- Axios
- React Router
- React Toastify
- Swiper

## Setup

- Clone the repository
- Install dependencies
- Set up environment variables
- Run the development server

## Usage

- Start the development server
- Access the application at `http://localhost:5173`
